DataService

192.168.1.188服务器

bind绑定接口

方法说明：将指定username的客服和客服类型的_index绑定起来

方法路径：http://192.168.1.188:8080/DataService/services/AccountBindService/bind

参数示例：

  • username:_webadmin-1-0E3C1B4A-FB15-8F8B-CEF1-72B5FDD49913
  • _index:3

返回类型：

  • 成功：{ “result”: “SUCCESS” }
  • 失败:{ “result”: “FAIL” }

setDefault设置默认客服接口

方法说明：设置指定username的客服为默认客服

方法路径：http://192.168.1.188:8080/DataService/services/AccountDefaultService/setDefault

参数示例：

  • username：_webadmin-1-0E3C1B4A-FB15-8F8B-CEF1-72B5FDD49913

返回类型：

  • 成功：{ “result”: “SUCCESS” }
  • 失败:{ “result”: “FAIL” }

客服类型add接口

方法说明：增加一种客服类型到客服类型字典表中

方法路径：http://192.168.1.188:8080/DataService/services/AccountTypeService/add

参数示例：

  • name:abc

返回类型：

  • 成功：{ “result”: “SUCCESS”, “data”: [] }
  • 失败:{ “result”: “FAIL”, “data”: [] }

客服类型delete接口

方法说明：删除指定_index的客服类型

方法路径：http://192.168.1.188:8080/DataService/services/AccountTypeService/delete

参数示例：

  • _index:9

返回类型：

  • 成功：{ “result”: “SUCCESS”, “data”: [] }
  • 失败:{ “result”: “FAIL”, “data”: [] }

客服类型更新update接口

方法说明：更新指定_index的客服类型名称

方法路径：http://192.168.1.188:8080/DataService/services/AccountTypeService/update

参数示例：

  • _index:8
  • name:abc

返回类型：

  • 成功：{ “result”: “SUCCESS”, “data”: [] }
  • 失败:{ “result”: “FAIL”, “data”: [] }

一维数据统计接口

  • table:trackuseragent
  • xaxis:os_name
  • ymode:count
  • target:_index
  • filter:city='北京'(可选参数)

{

打电话接口

方法说明：根据给定的caller和callee，先给caller打，接通之后给callee打

方法路径：http://192.168.1.188:8080/DataService/services/CallService/start

参数示例：

  • spid:559228
  • appid:343
  • caller:13163359533
  • callee:15627563706

返回类型： 

{
"result": "SUCCESS"
}

发送验证码接口

方法说明：向指定的phoneNumber发送验证码

方法路径：http://192.168.1.188:8080/DataService/services/SendVerification/sendVerificationCode

参数示例：

  • phoneNumber：13163359533

返回类型： 

{
"resutl": "SUCCESS"
}

查询消息发送次数接口

方法说明：查询模板消息发送次数

方法路径：http://192.168.1.188:8080/DataService/services/TemplateService/getTemplateQuery

参数示例：ids：[89,121]

返回类型： 

{
"total": 2,
"result": "SUCCESS",
"list": [

验证验证码接口
